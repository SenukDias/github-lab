## ✅ Intermediate Goals

By the end of these labs you will:

- [ ] Create and manage branches confidently
- [ ] Merge and rebase with understanding of the difference
- [ ] Open, review, and merge Pull Requests
- [ ] Fork and contribute to any repository
- [ ] Create a basic GitHub Actions workflow
