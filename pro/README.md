## ✅ Pro Goals

By the end of these labs you will be able to:

- [ ] Use cherry-pick, bisect, reflog, and worktrees
- [ ] Write complex `gh` CLI scripts and automations
- [ ] Create pre-commit, commit-msg, and post-merge hooks
- [ ] Build multi-stage CI/CD pipelines with caching and artifacts
- [ ] Configure branch protection, required reviews, and secrets scanning
