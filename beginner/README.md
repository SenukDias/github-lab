# 🟢 Beginner Labs — Overview

Welcome to the **Beginner Track**! These labs take you from zero to having your first repository on GitHub.

## Labs in This Section

| Lab | Topic | Time |
|-----|-------|------|
| [Lab 01](./lab-01-installation.md) | Installing Git & GitHub CLI | 15 min |
| [Lab 02](./lab-02-configuration.md) | Configuring Git Identity | 10 min |
| [Lab 03](./lab-03-github-login.md) | GitHub Authentication | 10 min |
| [Lab 04](./lab-04-first-repo.md) | Creating Your First Repository | 20 min |
| [Lab 05](./lab-05-basic-commands.md) | Essential Git Commands | 25 min |
| [Lab 06](./lab-06-gitignore.md) | The .gitignore File | 10 min |

## ✅ Beginner Goals

By the end of these labs you will be able to:

- [ ] Install and verify Git and the `gh` CLI
- [ ] Set your global Git identity
- [ ] Authenticate the `gh` CLI with your GitHub account
- [ ] Create a repository (both via web and CLI)
- [ ] Add, commit, and push files
- [ ] Pull changes from GitHub
- [ ] Write a proper `.gitignore`
